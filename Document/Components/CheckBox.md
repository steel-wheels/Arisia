# `CheckBox` component

## Introduction
Check box is used to select the parameter status (true or false).

## Interface

This is the interface definition for TypeScript:
<pre>
interface CheckBoxIF extends FrameIF {
  label : string ;
  status : boolean ;
}
declare function _alloc_CheckBox(): CheckBoxIF ;

</pre>
